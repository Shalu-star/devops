ABOUT ADDON:

addon cannot be used independently , it is used with some storefront.

CREATING ADDON
1. ant extgen
2. choose yaddon template
3. write the name of addon
4. write package name
5. add addon name in localextension.xml file of config folder (source), add target addon (eg; trainingstorefront), check whether addonsupport is present or not if not then add in the file
6.ant addoninstall -Daddonnames="hybrisTestAddon" -DaddonStorefront.storefrontTemplateName="trainingstorefront"
7. ant addonuninstall -Daddonnames="hybrisTestAddon" -DaddonStorefront.storefrontTemplateName="trainingstorefront" ---> for removing the addon from trainingstorefront
8. ant clean all
